## ✨ Overview

This repository is a personal Python practice workspace focused on learning core programming concepts through simple, hands-on exercises. It includes beginner-level scripts, data structure examples, and practice problems that help build confidence in Python.

This repository is organized into practical Python topics such as:

- Basic syntax and output
- Variables, operators, and conditions
- Loops and functions
- Lists, tuples, and dictionaries
- Recursion and problem-solving exercises
- Data structure examples using Python collections

## 📁 Project Structure

```text
01_basics/         # Basic Python programs and exercises
data-structure/   # Examples of lists, dictionaries, and collections
test/             # Practice files and small test programs
```

## 🚀 How to Run

Make sure Python 3 is installed, then run any script using:

```bash
python3 filename.py
```

Example:

```bash
python3 01_basics/01_hello-world.py
```

- Number and string operations
- Palindrome, prime, Armstrong, and GCD programs
- Vowel counting and word counting
